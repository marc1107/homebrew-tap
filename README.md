# Homebrew Tap

Custom Homebrew tap for marc1107's tools.

## Installation

```bash
brew tap marc1107/tap
```

## Available Casks

### WhisperType

Local speech-to-text dictation for macOS, powered by whisper.cpp.

```bash
brew install --cask whisper-type
```

For more info: [github.com/marc1107/whisper-type](https://github.com/marc1107/whisper-type)
